# ee241
## Programming for Signal and Information Processing Applications

This repository contains lab assignments and reports for the EE 241 course.

### Course Structure

- **Lab 1**: Introduction and Basic exercises
- **Lab 2**: Report and Exercises
- **Lab 3**: Image Processing
  - Includes sample images and exercises on image manipulation.
- **Lab 4**: Financial Data Analysis
  - Analysis of stock market data (DJI, GOOGL, TSLA).
- **Lab 5**: Connectome Dynamics (C. Elegans)
  - Analysis of the C. Elegans neural network (`dynworm` package).
  - Includes simulation and report templates.
